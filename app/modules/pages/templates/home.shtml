Code Modules — Sample Files

This is a silly sample website built using code modules techniques. Code Modules hasn’t been released yet — we’re aiming at August 1, 2021. When released, the open source files for building this site can be accessed at GitHub.

Code Modules is built upon the use of BBEdit’s include system which is a dependency. If you don’t wish to use BBEdit, a paid program, Code Modules won’t work for you.

Top Level Module Types

There are five top level module types. Each of the items below are linked to the documentation for that type.

This site demonstrates a website built using Code Modules. The source code for this website is available on GitHub. Documentation for Code Modules is on a separate website.
